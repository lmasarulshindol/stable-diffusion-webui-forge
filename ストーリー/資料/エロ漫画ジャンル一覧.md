# エロ漫画ジャンル一覧・属性リスト

## 1. 関係性・雰囲気 (王道〜背徳)
* **純愛 (Vanilla / Pure Love):** 相思相愛。甘い雰囲気。
* **イチャラブ:** 純愛よりもさらに行為そのものを楽しみ、イチャイチャすることに重点を置いたもの。
* **ハーレム (Harem):** 一人の男性が多数の女性に愛される。
* **逆ハーレム:** 一人の女性が多数の男性に愛される。
* **近親相姦 (Incest):** 母、姉、妹、娘など血縁者との関係。
* **寝取り (NTR / Netorare):** パートナーを他人に奪われる。
    * **寝取らせ:** 自分のパートナーをあえて他人に抱かせる。
    * **寝取り (NTL / Netori):** 他人のパートナーを自分が奪う。
    * **BSS (僕が先に好きだったのに):** 付き合ってはいないが、好意を寄せていた相手が他人に奪われる。

## 2. シチュエーション・設定 (特殊・SF・ファンタジー)
* **催眠 (Hypnosis / Mind Control):** 催眠術や魔法で常識改変や隷属化を行う。
* **時間停止 (Time Stop):** 時間を止めて好き放題する。
* **透明人間 (Invisibility):** 透明になって悪戯や行為を行う。
* **入れ替わり (Body Swap):** 男女の人格が入れ替わる。または皮を被ってなりすます（皮モノ）。
* **憑依 (Possession):** 体を乗っ取る。
* **TS (Gender Bend):** 性転換。男→女、女→男への肉体変化。
* **異種姦 (Interspecies):** モンスター（オーク、ゴブリン、触手など）との行為。
* **風俗 (Brothel / Sex Work):** ソープ、デリヘルなどの設定。
* **公衆便所 (Public Use):** 不特定多数に扱われる扱い。

## 3. 強制・ハード系
* **レイプ (Rape / Non-con):** 同意のない強制的な行為。
* **和姦 (Consensual):** 合意の上での行為（純愛に近いが、ジャンルとして区別される場合がある）。
* **調教 (Training / BDSM):** 支配と服従。SM、拘束、開発など。
* **薬漬け (Drugs / Aphrodisiac):** 媚薬やドラッグで理性を飛ばす。
* **洗脳 (Brainwashing):** 精神を書き換えて、相手に従順になるようにする。堕ちる過程を楽しむもの。
* **肉体改造 (Body Modification):** 感度倍増や身体機能の拡張など。

## 4. キャラクター属性・ヒロインタイプ
* **人妻・熟女 (MILF):** 母性や包容力、あるいは背徳感。
* **ギャル (Gyaru):** 見た目は派手だが純情、あるいは奔放など。黒ギャル含む。
* **メスガキ:** 生意気な言動で相手を煽る少女（または小柄な女性）。わからせ（お仕置き）とセットになることが多い。
* **ツンデレ:** 普段はツンツンしているがデレる。
* **ビッチ (Bitch / Slut):** 貞操観念が低い、または性欲が強い。
* **処女 (Virgin):** 初めての行為。
* **お姉さん (Onee-san):** 年上の頼れる女性、誘惑してくる隣人など。
* **女教師:** 学校ものでの定番。
* **ふたなり (Futanari):** 女性の体に男性器がついている。
