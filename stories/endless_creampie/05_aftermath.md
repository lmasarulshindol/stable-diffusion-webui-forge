# Scene 5: 白い廃墟

最後の男が咆哮と共に果て、その肉棒を引き抜いた時、スタジオには重苦しい静寂が訪れた。
聞こえるのは、荒い息遣いと、ポタポタと液体が滴る音だけだった。

「お疲れ……」

誰かが小さく呟いたが、その声に応える者は誰もいなかった。
男たちは皆、やりきったという虚脱感と共に、それぞれの服を拾い上げていた。
そして、その中央には、一人の女が残されていた。

水原愛衣。
彼女は、白いビニールシートの上で大の字になって倒れていた。
その姿は、まるで嵐が過ぎ去った後の廃墟のようだった。

純白だったキャミソールは、見る影もなく変色していた。
汗、涎、そして大量の精液が染み込み、半透明になって肌に張り付いている。破れた箇所からは、赤いキスマークや手の跡が無数に刻まれた肌が覗いていた。
かつて輝くようだった黒髪は、今はベトベトに固まり、ワカメのように床に散らばっている。

しかし、最も凄惨かつ背徳的なのは、彼女の下半身だった。
大きく開かれたままの足の間には、白い池ができていた。
秘部は赤黒く腫れ上がり、だらしなく開いたまま閉じることができなくなっている。そこからは、呼吸をするたびに「ゴボッ」という音と共に、白濁した液体が溢れ出し続けていた。
お腹は異様なほどに膨らんでいた。
何人分もの精液を注ぎ込まれた子宮は、まるで妊娠したかのようにポッコリと突き出し、その存在感を主張していた。

「愛衣ちゃん……大丈夫？」

スタッフが恐る恐る声をかけ、カメラを近づけた。
愛衣の目は、天井の一点を見つめたまま動かなかった。
瞳孔が開いたその瞳には、光が宿っていないように見える。焦点が合っておらず、虚空を彷徨っていた。

「あ……ぅ……」

唇が微かに動いた。
乾ききった喉から漏れたのは、言葉とは呼べない掠れた音だけだった。
口の端からは、拭われることのない涎が、糸を引いて頬を伝い落ちている。舌は力なく口の外にはみ出したままだ。

身体は、まだ微かに痙攣していた。
指先がピクピクと動き、太腿の内側の筋肉が時折波打つように収縮する。
それは、過剰なまでの快楽を受け続けた肉体が、まだその余韻から抜け出せていない証拠だった。
彼女の神経は、まだ終わらない性交の最中に取り残されているのかもしれない。

カメラマンが、彼女の顔のアップを撮るためにレンズを近づけた。
その気配に気づいたのか、愛衣の視線がゆっくりとカメラの方へ向いた。
一瞬、その瞳に光が戻ったように見えた。

「……んふ」

彼女は笑った。
それは、廃人のような虚ろな笑みでありながら、どこか満ち足りた、聖母のような慈愛さえ感じさせる不可思議な表情だった。
理性が完全に崩壊し、獣としての本能だけが残った女の、到達点の顔。

「いっぱい……入ってる……」

彼女は震える手で、自分のお腹を愛おしそうに撫でた。
ポッコリと膨らんだ下腹部。そこには、男たちの欲望のすべてが詰め込まれている。
その感触を確かめるように、彼女は何度も何度もお腹を擦った。

「あったかい……お腹、重くて……幸せ……」

その言葉に、周りのスタッフたちは息を呑んだ。
彼女は壊れたのではない。
新たな形へと「生まれ変わった」のだ。
清楚なアイドルとしての水原愛衣は、この白い部屋で死んだ。そして今、快楽の奴隷としての新しい水原愛衣が誕生したのだ。

「もう……普通のセックスじゃ、満足できないね」

ディレクターが独り言のように呟いた。
その言葉は、残酷な予言のように響いた。
これほどの刺激、これほどの陵辱、これほどの快楽を知ってしまった彼女が、以前のような日常に戻ることなど不可能なのだから。

愛衣は、再び天井を見上げた。
白い天井が、眩しい。
身体中の穴という穴がヒリヒリと痛み、感覚がないほどに痺れている。
けれど、その痛みさえもが、愛おしかった。

（また……したい……）

薄れゆく意識の中で、彼女はそう願った。
もっと激しく、もっと汚く、もっと深く。
底なしの沼に沈んでいくような感覚に身を委ねながら、愛衣はゆっくりと目を閉じた。

彼女の股間からは、まだ止めどなく白い液体が流れ続けていた。
それは、終わらない物語の、終わりのないエピローグのようだった。

カットの声がかかることはなかった。
カメラは、彼女が眠りに落ちるその瞬間まで、その無惨で美しい姿を記録し続けていた。
白いスタジオに残された、白濁にまみれたヴィーナス。
その映像は、伝説として語り継がれることになるだろう。

＜完＞
